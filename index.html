Guía de inicio
1
Espera que cargue
El servidor puede tardar ~30 seg si estuvo inactivo. Es normal en el plan gratuito de Render.
2
Haz clic en la terminal
Da clic dentro del área negra para activar el cursor y empezar a escribir.
3
Prueba un comando
Empieza con neofetch para ver la info del sistema.
4
Instala lo que quieras
Usa sudo apt install para instalar cualquier paquete de Ubuntu.
5
⚠ Todo se borra al reiniciar
El plan gratuito de Render no guarda cambios entre reinicios. Úsalo para practicar.
